## 🛠 Предлагаемый технологический стек

Рекомендую вариант, который даёт современный вид, быструю загрузку, простое
добавление публикаций и **бесплатный хостинг**.

### Рекомендуемый (простой и современный)
| Компонент | Технология | Почему |
|-----------|-----------|--------|
| Фреймворк | **Astro** | Очень быстрый, идеален для контентных сайтов, минимум JS |
| Стили | **Tailwind CSS** | Быстрая современная вёрстка, готовые адаптивные утилиты |
| Контент/публикации | **Markdown + Content Collections** | Каждая работа — отдельный файл, легко добавлять |
| Хостинг | **GitHub Pages / Netlify / Vercel** | Бесплатно, автодеплой при обновлении |
| Домен | Reg.ru / Namecheap | Свой адрес для солидности |

### Альтернатива 1 — если нужна интерактивность
- **Next.js (React)** + Tailwind CSS + Vercel — мощнее, но сложнее в поддержке.

### Альтернатива 2 — без программирования
- **WordPress** с академической темой — можно вести самому через админку, но менее «современно» и требует хостинга с БД.

> **Мой совет:** начать с **Astro + Tailwind + GitHub Pages**. Это оптимально
> для научного сайта-визитки: современно, быстро, бесплатно и легко пополнять.
